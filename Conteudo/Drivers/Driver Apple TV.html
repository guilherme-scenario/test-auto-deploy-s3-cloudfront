Driver Apple TV

Ao utilizar o driver de integração Embrace para a Apple TV, o usuário poderá controlar o dispositivo de TV a partir da E-App e dos demais recursos de automação que a plataforma Embrace oferece.

O driver conta com um template para E-App que, além de englobar todas as funcionalidades presentes no controle remoto próprio do aparelho, também oferece uma facilidade de navegação na forma de atalhos para acesso rápido a todos os aplicativos instalados em seu dispositivo Apple TV.

A partir das Conexões de Entrada e Saída do driver de integração, também é possível customizar o projeto Embrace para programar acionamentos e controlar o dispositivo de TV por meio de Atividades, Macros, Eventos e todos os demais recursos de automação oferecidos pela plataforma Embrace.

A integração com Apple TV é oferecida a partir dos aparelhos Geração 4, e para integrar o serviço ao sistema Embrace, basta utilizar o Config E2 para incluir o driver de integração ao projeto (e programar as Atividades de áudio e vídeo que serão responsáveis por gerenciar os acionamentos do dispositivo) e a E-App para realizar o processo de pareamento com o aparelho da Apple.

E-Config

Conforme mencionado, o driver de integração para Apple TV precisa estar inserido no projeto Embrace para que ele possa realizar o controle dos recursos do aparelho, processo realizado através do software Scenario Config Embrace.

Para inserir o driver no projeto, basta acessar a guia de Drivers no menu de Equipamentos e utilizar a busca para encontrar um dispositivo do tipo Media Player com o nome de APPLE TV. Alternativamente, também é possível navegar pelos diretórios da árvore de drivers até Scenario > Media Player > Scenario > APPLE TV.

Uma vez que o driver esteja devidamente inserido no projeto Embrace, basta realizar a programação das Atividades de Áudio e Vídeo que serão responsáveis por gerenciar o controle do dispositivo, realizando as definições de Fonte e Volume de acordo com o setup da instalação (ao selecionar o dispositivo Apple TV como Fonte de uma Atividade, por exemplo, será possível acessar o template por meio desta Atividade) e enviar o projeto para a controladora.

E-App

Para ter acesso às funcionalidades da integração com a Apple TV, é necessário primeiramente realizar o pareamento entre o aparelho e o dispositivo do projeto por meio da E-App.

Após adicionar o driver de integração ao projeto Embrace, basta acessar a as Configurações do projeto por meio do ícone de Menu (canto superior direito).

Em seguida, deve-se acessar a tela de Serviços de integração, que listará todos os dispositivos de integração configuráveis presentes no projeto e, dentre eles, o Apple TV (que será exibido na lista de acordo com o nome que lhe foi atribuído durante o desenvolvimento do projeto).

  • Ao pressionar o campo Dispositivo, uma busca local será realizada e retornará uma lista com o nome e o endereço IP dos aparelhos encontrados na rede local para que o usuário selecione um deles a fim de finalizar o processo de Associar o dispositivo.
  • Ao selecionar o dispositivo escolhido, o processo de Autenticação será iniciado. O processo de Autenticação consiste em 3 etapas. Para cada uma delas ser completada, o usuário deverá sincronizar os dispositivos digitando na E-App o PIN numérico com 4 valores que será exibido na tela conectada ao aparelho da Apple TV assim que o processo de Autenticação for iniciado.

Template

Com o dispositivo de integração devidamente inserido no projeto, programado e pareado com o aparelho, será possível acessar o template e utilizar todas as funções do controle por meio qualquer Atividade de Áudio e Vídeo convencional que defina o Apple TV como Fonte.

O template foi pensado para que seu layout reflita o do controle remoto próprio do aparelho, facilitando assim a intercambialidade no uso dos dispositivos e a preservação dos recursos disponíveis no equipamento original.

Dentre os recursos disponíveis, pode-se citar o painel touch, utilizado para navegação geral e para controle fino da barra de progresso do conteúdo que está sendo reproduzido, e também o menu de acesso rápido aos aplicativos, que é responsável por listar atalhos para todas as apps atualmente instaladas no aparelho.
