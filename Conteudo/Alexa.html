Alexa

A Embrace oferece a possibilidade de integração com a Amazon Alexa, assistente virtual da Amazon, que reconhece comandos de voz e os executa. Dessa forma, tudo que o usuário faria utilizando botões físicos ou o aplicativo Embrace será possível apenas utilizando os comandos de voz. Para utilizar essa funcionalidade é preciso baixar o aplicativo oficial, disponível na App Store.

Ao abrir o aplicativo será solicitado o acesso utilizando uma conta. Se já tiver uma conta e quiser utilizá-la basta efetuar login. Caso contrário é preciso criar uma conta.

Será aberta uma tela para aceitar os termos de uso da funcionalidade Alexa.

A partir daí será possível escolher a partir de qual equipamento deseja enviar comandos de voz: Echo, Echo Dot, Echo Plus ou Amazon Tap.

Toque em Get Started para finalizar cadastro.

Selecione seu usuário.

Adicione todos os dados necessários e será exibida a seguinte tela:

A partir daí será possível adicionar a funcionalidade Embrace. Para isso, basta acessar a seção de Skills do aplicativo. Será aberta uma tela contendo diversas opções. Em Search all skills digite Embrace e toque na opção mostrada a seguir.

Toque em Enable para habilitar a Embrace.

Será aberta uma tela de login de usuário da Embrace.

A seguinte tela listará as instalações às quais o usuário informado no passo anterior tem acesso.

Finalizando a integração com a Embrace, toque em OK localizado no canto superior esquerdo da tela e em seguida selecione a opção Discover devices para que seja feita uma busca de dispositivos.

Neste momento será possível controlar absolutamente tudo que a Embrace oferece desde iluminação até atividades de áudio e vídeo.

Abaixo estão listados alguns comandos que podem ser utilizados para realizar o acionamento do dispositivos da automação que foram integrados à Assistente.

Também é possível obter um relatório de comandos de voz válidos para o seu projeto utilizando o recurso de Exportar Comandos de Assistente Virtual no E-Config.

Comandos suportados:

Discovery Message

Usada para descobrir quais os devices e cenas associadas a conta do usuário:

Comando

"Alexa, discover my smart home devices"."Alexa, descobrir meus dispositivos".

ou pela função discovery no aplicativo Alexa.

Comandos para ativar cenas/atividades

Comando

"Alexa, turn on scene <scene_name>""Alexa, turn on <activity_name>""Alexa, ligar cena <scene_name>""Alexa, ligar <activity_name>"

Comandos para dispositivos de iluminação

Usado para controlar a dimerização de uma (ou mais) lâmpadas.

Comando

"Alexa, turn on/off <device_name>""Alexa, ligar/desligar <device_name>""Alexa, acender/apagar <device_name>""Alexa, dim <device_name> to <brightness_delta_value>""Alexa, aumentar <device_name> para <brightness_delta_value>""Alexa, set the <device_name> to <brightness_value>""Alexa, definir o <device_name> para <brightness_value>"

Comandos para dispositivos de iluminação RGB

Usado para controlar a cor de uma lâmpada.

Comando
