Barra de menus

A Barra de menus é composta pelas guias de Arquivo, Preferências e Sistema.

Arquivo

  • Novo: Cria um novo projeto;
  • Abrir: abre projeto a partir de pastas;
  • Recentes: abre projetos que foram abertos recentemente;
  • Salvar: salva projeto atual;
  • Salvar como: salva uma cópia do projeto atual e passa a editar esta nova cópia;
  • Sair: Sai do projeto e finalizado o E-Config.

Preferências

Preferências do Sistema

  • Salvar Automaticamente: define o intervalo de tempo que o sistema usará para avisar o usuário para salvar o projeto;

  • Opções envio projeto: o usuário pode escolher se deseja que a iluminação seja reiniciada ao enviar projeto;

  • Opções Modo Teste: ao sair do modo teste o estado dos módulos e teclados podem ser mantidos ou limpos (voltam ao estado anterior).

Preferências do Projeto

  • Opções de Instalação
    • Assistente virtual em inglês: Permite alternar a língua utilizada na integração com Assistente Virtual (PT-BR/EN): também altera o Relatório de Comandos de Voz;
    • Controle volume exclusivo por atividades: Habilita/desabilita o controle de volume de dispositivos de áudio/vídeo por meio de Atividades ou diretamente pelo template do driver;
    • Habilitar Circuito: Habilita a visualização do identificador atribuído a circuitos na árvore do projeto (exibe nome do circuito entre colchetes na frente do nome do circuito);
  • Opções de Projeto Gráfico
    • Omitir menu Inteligência para usuários na E-App: Habilita/desabilita a visualização das opções de Inteligência para o projeto na E-App;
    • Nome Projeto gráfico completo: Habilita/desabilita a visualização de nomes contendo a estrutura completa de ambientes (e seus sub-ambientes) na E-App;
    • Habilitar paginação nos templates de projeto gráfico: Habilita/desabilita a paginação de templates de drivers do projeto;
  • Opções de Integração
    • Habilita/desabilita opções para integrações com terceiros;
  • Opções de Log
    • Habilita/desabilita opções de Logs da Controladora;

Sistema

  • Procurar Atualização: Faz uma busca nos servidores buscando novas versões (esta busca já é feita na inicialização do sistema, mas caso não tenha disponibilidade de internet na abertura do mesmo, isso pode ser feito posteriormente escolhendo esta opção).
  • Exportar KPView: o KPView possibilita criar labels (legendas) para teclas de qualquer teclado, dentre outras possibilidades. Essa exportação pode ser feita de duas formas:
    • PDF: exporta simulando a aparência do teclado com os labels e não é possível editar no momento da impressão, apenas pelo sistema;
    • TXT: exporta apenas os nomes em arquivo com extensão .txt para não ter problemas de compatibilidade de editor de texto e ser possível editar no momento da impressão.
  • Exportar Circuito de Iluminação: Como forma de auxílio na localização das luminárias na rede física, o E-Config disponibiliza um documento contendo módulo de iluminação, luminária e circuito de cada luminária de cada ambiente, respectivamente. Os circuitos podem ser vistos na própria árvore do projeto, para isso é preciso selecionar a opção Habilitar Circuito, localizado em Instalação, Configurações, Opções de instalação.
  • Exportar Comandos Assistente Virtual: Para utilizar comandos de voz na integração com Assistente Virtual, é preciso usar comandos que ela consiga interpretar para executá-los. Por isso, o E-Config disponibiliza um documento contendo os comandos necessários para que o usuário consiga controlar toda a instalação.
  • Verificar Projeto: Executa o Verificador de Pendências no projeto;
  • Ajuda: Redireciona para o Help para que o usuário possa sanar qualquer dúvida envolvendo a linha Embrace.
  • Sobre: Redireciona para a página de Releases, ou seja, correções e implementações feitas a cada versão de E-Config e Controladora.
